 ## 💰 CONVERT MONEY - COMPLETE
<br>
> Conversor de moedas completo desenvolvido com HTML, CSS e JavaScript.
<br>


## 📌 SOBRE O PROJETO
<br>
<p>
 O **CONVERT MONEY** é um conversor de moedas que permite realizar conversões entre diferentes moedas de forma rápida, prática e intuitiva.
Diferente de um conversor simples, este projeto permite converter **qualquer moeda para qualquer outra moeda disponível no sistema** 💱🔄.
O projeto foi desenvolvido com foco na consolidação de conhecimentos em desenvolvimento front-end, utilizando apenas tecnologias base, sem frameworks ou bibliotecas externas. 
</p>
<br>

## FUNCIONALIDADES 🎯
<br>

✔ Conversão entre as seguintes moedas:

- 🇧🇷 Real (BRL)
- 🇪🇺 Euro (EUR)
- 🇺🇸 Dólar Americano (USD)
- 🇬🇧 Libra Esterlina (GBP)
- ₿ Bitcoin (BTC)

✔ Conversão de **todas para todas**
- Real → Dólar
- Dólar → Euro
- Euro → Bitcoin
- Libra → Real
- Bitcoin → Dólar
- E qualquer outra combinação possível

✔ Cálculo automático e instantâneo  
✔ Atualização dinâmica ao alterar valor ou moeda  
✔ Campo livre para inserir qualquer valor  
✔ Interface simples e intuitiva  
<br>
